Project Introduction

Uni10(Universal Tensor Network Library) is programmed in C++ and geared toward applications to the tensor network algorithms, such TEBD, PEPS and MERA. We hope the availability of Uni10 will make future implementation of tensor network algorithms much easier and less error prone.

Out Contribution
